== Description ==

Mobile Menu CK is the perfect solution to create your mobile menus. Create an amazing mobile menu that looks like a mobile application and give your users a great mobile experience. It is compatible with many menus and can be styled directly whithin the interface.

= Features =
* Device detection
* Multi compatibility
* Styling interface
* Options integration
* Multiple presets
* No limit
* Touch ready and user friendly
* Can be used by developpers
* Multiple effects to animate your menu


== Installation ==

= Using the WordPress Plugin Search =

1. Navigate to the `Add New` sub-page under the Plugins admin page.
2. Search for `mobile menu ck`.
3. The plugin should be listed first in the search results.
4. Click the `Install Now` link.
5. Lastly click the `Activate Plugin` link to activate the plugin.

= Uploading in WordPress Admin =

1. [Download the plugin zip file](http://wordpress.org/plugins/mobile-menu-ck/) and save it to your computer.
2. Navigate to the `Add New` sub-page under the Plugins admin page.
3. Click the `Upload` link.
4. Select Easy Table of Contents zip file from where you saved the zip file on your computer.
5. Click the `Install Now` button.
6. Lastly click the `Activate Plugin` link to activate the plugin.

= Using FTP =

1. [Download the plugin zip file](http://wordpress.org/plugins/mobile-menu-ck/) and save it to your computer.
2. Extract the Mobile Menu CK zip file.
3. Create a new directory named `mobile-menu-ck` directory in the `../wp-content/plugins/` directory.
4. Upload the files from the folder extracted in Step 2.
4. Activate the plugin on the Plugins admin page.
